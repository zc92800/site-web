About Agora Cook

Agora Cook is a recipe website that allows users to view a collection of recipes and see detailed information for each one. The main page loads recipes automatically when the user opens the site.

The search bar allows users to search for specific recipes by name. When the user clicks on a recipe, the site shows the ingredients and instructions for that recipe.

The site uses the Spoonacular API to get recipe data. This means it fetches information from an external source instead of storing it directly in the website.

The code structure is simple:

  • index.html - the main page with the search bar and recipe display.
  • index.css - styles for the page, including recipe cards and layout.
  • app.js - JavaScript code that handles fetching recipes, searching, and displaying details.

The site is designed to be responsive, so it works on both desktop and mobile devices.
